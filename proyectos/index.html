Proyecto Dockers

¿Qué es un Docker?

Docker es una plataforma de código abierto que permite crear, empaquetar, desplegar y ejecutar aplicaciones en contenedores. Un contenedor Docker encapsula una aplicación y todas sus dependencias, pero comparte el kernel con el sistema operativo anfitrión, lo que lo hace muy ligero y eficiente en comparación con las máquinas virtuales tradicionales. Gracias a Docker, puedes desplegar aplicaciones y servicios de forma rápida, portable y reproducible, optimizando recursos, controlando versiones y facilitando el trabajo en la nube, en entornos locales o compartiendo imágenes en repositorios públicos o privados.

Requesitos minimos para poder hacer un docker

Requisitos generales de hardware
Procesador: 64 bits con soporte de virtualización (Intel VT-x, AMD-V).
Memoria RAM: Al menos 2 GB para pruebas básicas, pero se recomienda 4 GB o más para un funcionamiento fluido.
Espacio en disco: 10-20 GB libres como mínimo para imágenes y contenedores; más si vas a trabajar con proyectos grandes o muchas imágenes.

Guia de Comandos basicos

Comando Descripción Ejemplo
docker pull <imagen> Descarga una imagen del registro remoto. docker pull ubuntu:latest
docker run <opciones> <imagen> Crea y ejecuta un contenedor a partir de una imagen. docker run -it ubuntu /bin/bash
docker ps Lista los contenedores en ejecución. docker ps
docker ps -a Lista todos los contenedores (activos y detenidos). docker ps -a
docker images Muestra las imágenes disponibles localmente. docker images
docker stop <contenedor> Detiene un contenedor en ejecución. docker stop webserver
docker rm <contenedor> Elimina un contenedor (debe estar detenido). docker rm webserver
docker rmi <imagen> Elimina una imagen local. docker rmi ubuntu:latest
docker exec -it <contenedor> <comando> Ejecuta un comando dentro de un contenedor en ejecución. docker exec -it webserver bash
docker logs <contenedor> Muestra los logs de salida del contenedor. docker logs webserver
docker build -t <nombre> <ruta> Construye una imagen desde un Dockerfile. docker build -t miapp .
docker network ls Lista las redes definidas en Docker. docker network ls
docker volume ls Lista los volúmenes creados en Docker. docker volume ls
docker system prune Limpia recursos no usados (contenedores, imágenes, volúmenes). docker system prune
